# 生日礼物网页

这是一个可直接放进 GitHub 仓库的静态网页骨架。

## 当前文件

- `index.html`：页面结构
- `styles.css`：页面样式

## 后续替换位置

1. 把 `index.html` 里的占位文案替换成正式内容。
2. 把视频文件放到 `assets/birthday-highlight.mp4`。
3. 如果要放封面图，也可以后面给 `video` 标签加 `poster`。

## 发布方式

直接把整个 `生日礼物` 文件夹内容推到 GitHub 仓库根目录，或作为单独仓库发布到 GitHub Pages 即可。
